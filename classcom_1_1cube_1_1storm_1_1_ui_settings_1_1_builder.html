Detailed Description

The builder class for com.cube.storm.UiSettings. Use this to create a new com.cube.storm.UiSettings instance with the customised properties specific for your project.

Call build() to build the settings object.

Builder imageLoaderConfiguration ( ImageLoaderConfiguration.Builder  configuration)

Sets the default image loader configuration.
