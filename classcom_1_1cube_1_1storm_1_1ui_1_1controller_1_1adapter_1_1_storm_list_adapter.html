Detailed Description

The base adapter used for displaying Storm views in a list. Using an adapter to do such a task has the benefit of view recycling which makes the content smooth to scroll.

This adapter only supports com.cube.storm.ui.model.Model classes which have a defined com.cube.storm.ui.view.holder.Holder counter-class.

Usage

Problems Problems can arise with this method of rendering content, specifically with render efficiency where the views are not being recycled because there is only 1 of its view type in the list. This is the equivalent of having all of the views inflated into a android.widget.ScrollView. The smoothness of the scrolling (depending on how much content there is) diminishes with the amount of unique content that the list is rendering.
